# EventApp - FullStack
BackEnd: Node.js + MongoDB
FrontEnd: React Native

# Back-End Usage
- Node_module package installed, no need to download again!
- Run `npm run dev` to start your application!

## Back-End Base dependencies
- `Typescript`
- `ES6`
- `Express`
- `Swagger`
- `Mongoose`
- `Dotenv`
